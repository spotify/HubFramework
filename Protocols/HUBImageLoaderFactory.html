HUBImageLoaderFactory

@protocol HUBImageLoaderFactory <NSObject>

Protocol that objects that create image loaders for use with the Hub Framework conform to

You conform to this protocol in a custom object and pass that object when setting up HUBManager. The Hub Framework will then use the factory to create an image loader for each view controller that it creates. See HUBImageLoader for more information.

  • -createImageLoader

    Create an image loader

    This will be called every time that a view controller is created by the Hub Framework

    Declaration

    Objective-C

    - (nonnull id<HUBImageLoader>)createImageLoader;
